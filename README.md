# TMDB-Movie-Data-Analysis
This project analyzes the TMDB movie dataset to uncover trends in movie popularity, revenue, genres using Python-based data analysis and visualization techniques.  The objective of this project is to clean, explore, and visualize movie data to generate meaningful insights that can help understand factors influencing movie success.


# The dataset contains information about movies such as:
Title
Genre
Budget
Revenue
Popularity
Release Year
Cast
Runtime
Source: TMDB Movies Dataset

# Key Tasks Performed
Data Cleaning and Preprocessing
Handling Missing Values
Feature Exploration and Transformation
Genre-based Analysis
Popularity and Revenue Distribution Analysis
Data Visualization

# Exploratory Data Analysis
The analysis focuses on:
Distribution of movie popularity
Most common movie genres
Relationship between budget and revenue
Trends in movie releases over the years
High-performing movies based on revenue and popularity

# Technologies Used
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
VS Code

# Sample Insights
Certain genres consistently produce higher revenue.
Movies with higher budgets tend to have higher revenue potential.
Popularity distribution shows that a small number of movies dominate audience attention.
Movie production has increased significantly over the years.
